##  命令  

1. 需要安装  node 7.2以上版本 npm 5.6以上版本  
2. 安装依赖： 项目根目录运行  npm install  
3. 开发启动： npm run dev   打开ip:9528   
4. 打包编译： npm run build   dist目录下为编译后的静态资源
